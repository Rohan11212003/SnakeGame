Assignment 4 - Snake Game

The objective of the game is to control the snake to eat food, grow longer, and avoid colliding with walls or itself.

  • Press Start button to start game.
  • Use the arrow keys (up, down, left, right) to control the direction of the snake.
  • Press the up arrow key to move the snake upwards.
  • Press the down arrow key to move the snake downwards.
  • Press the left arrow key to move the snake to the left.
  • Press the right arrow key to move the snake to the right.
  • If snake hits the border then it will die
  • If you want to restart the game click start button.
